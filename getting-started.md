# Getting Started

## Prerequisites

- **Node.js** 18+
- **NestJS** 10 or 11
- **Prisma** 5 or 6
- **PostgreSQL** 14+

## Quick Start

Install your first nestarc module:

```bash
npm install @nestarc/tenancy
```

## Stack Overview

All nestarc packages share a common stack:

- **NestJS** — progressive Node.js framework
- **Prisma** — type-safe ORM
- **PostgreSQL** — primary database with RLS support
- **TypeScript** — end-to-end type safety
